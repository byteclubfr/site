Formation ECMAScript 2015 (ES6)

Présentation

2 jours pour découvrir le JavaScript moderne : ECMAScript 2015.
Stage pratique, magistral et didactique, pour devenir autonome.

Objectifs :

    À l'issue de cette formation, les participants seront en mesure de :
  • Comprendre quand et comment tirer parti des nouvelles fonctionnalités du langage, notamment pour les traitements asynchrones
  • Mettre en place un environnement de développement ES6 compatible tous environnements,
  • Développer une application utilisant ces derniers ajouts.

Méthode pédagogique :

Alternance de 50% de cours magistraux et de 50% de travaux pratiques (live coding) sur machine individuelle. Les travaux pratiques nombreux permettent de mettre en pratique les concepts abordés au fur et à mesure. Un support de cours est remis à chaque participant au format électronique.

Public :

  • Chefs de projet Web,
  • Architectes techniques,
  • Développeurs JavaScript front-end ou back-end.

Pré-requis : Bonne connaissance de JavaScript, Connaissance HTML, pratique du développement web.

Au sommaire :

  • Les nouvelles syntaxes (destructuring, raccourcis, valeurs par défaut, paramètres restant, etc.) ;
  • Les classes ;
  • Les modules ;
  • L'asynchrone : promesses, générateurs et coroutines ;
  • L'environnement de développement ES6 : transpilation avec Babel, npm, et Webpack
