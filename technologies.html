Node.js, c’est quoi ?

Framework open-source basé sur le moteur V8 de Chrome, Node.js permet d'exécuter du JavaScript côté serveur, jusque là réservé côté client.

Node.js un interpréteur JavaScript exécutable, enrichissant le langage avec sa propre API.

Les avantages de Node.js

Sa spécificité vient de son API, entièrement orientée vers le non bloquant, qui permet d’écrire des applications avec d’excellents temps de réponse.

Node.js rend les services web en temps réel redoutables d'efficacité.

AngularJS, c’est quoi ?

AngularJS est un framework Javascript maintenu par Google.

AngularJS tire parti et étend les possibilités nouvelles offertes par html5.

Les avantages d’AngularJS

En découplant la manipulation du DOM de la logique métier, et grâce à une organisation très modulaire, vous pourrez concevoir des applications web riches.
